# Estudo_dirigido_PDS_parte03
Este repositório foi desenvolvido para fins acadêmicos na disciplina de Processamento Digital de Sinais (PDS). O objetivo é apresentar estudos teóricos e simulações computacionais sobre análise de sinais no domínio da frequência, utilizando DFT, FFT e Transformada-Z em aplicações práticas de engenharia e sistemas telemáticos.
